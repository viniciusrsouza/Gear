# Gear

- [x] Logging system
- [x] Cross-platform windowing (GLFW)
- [x] Event system
- [x] Input system
- [x] OpenGL context
- [ ] 2D (?)
- [ ] 3D Renderer
- [ ] 3D Physics
- [ ] Audio

More to be added as the project progresses.
